# Tarea-2-Regresion-Lineal-Multiple-Predecir-Precio-de-Vivienda-.
Predicción de precios UF
Script en Python que usa regresión lineal múltiple para estimar precios de departamentos. El modelo entrena con datos de superficie, piezas y distancia al metro para predecir el valor final. Usa scikit-learn para el modelamiento y calcula métricas de error como MAE y R2 para validar los resultados del test set.

Librerías usadas: pandas, numpy, scikit-learn.
